# Asset Management App

## How to Start

### 1. Database Setup
1. Ensure PostgreSQL is running.
2. Create database: `CREATE DATABASE asset_manager_db;`
3. Update `{project_name}/server/.env` with your DB password.

### 2. Start Backend
```bash
cd server
npm install
npm run dev
```

### 3. Start Frontend
Open a new terminal:
```bash
cd client
npm install
npm run dev
```
